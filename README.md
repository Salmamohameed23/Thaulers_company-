# ⚡ TOUGH HAULERS TRADE LIMITED

A premium industrial-grade energy website built with React and Tailwind CSS, representing a global engineering company specialized in solar power systems and energy storage solutions.

---

## 🌍 About The Project

This website represents **TOUGH HAULERS TRADE LIMITED**, a global provider of:

- Solar Power Systems
- Battery Energy Storage Systems (BESS)
- Full EPC Solar & Energy Engineering
- China-based supply chain solutions
- Engineering support in Egypt
- Projects from kW scale to multi-Gigawatt scale

The goal of this project is to deliver a **high-end, modern, industrial-style website** similar to top global energy companies.

---

## 🧱 Tech Stack

- React.js
- Tailwind CSS
- Framer Motion (animations)
- React Router
- Lucide Icons

---

## ✨ Features

- ⚡ Premium dark industrial UI
- 🎬 Smooth animations using Framer Motion
- 📱 Fully responsive design
- 🧩 Modular component-based architecture
- 🌍 Global-scale branding experience
